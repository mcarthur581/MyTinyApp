# MyTinyApp

A simple web application for user management and note-taking.

## Features
- User authentication
- Create, edit, and delete notes

## Tech Stack
- Python
- SQLite
- HTML/CSS

## How to Run

1. Clone the repository:
git clone https://github.com/YOUR-USERNAME/MyTinyApp.git

2. Go to the folder:
cd MyTinyApp

3. Run the app:
python app.py
